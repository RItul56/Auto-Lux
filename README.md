# 🚗 AutoLux - Luxury Car Showroom Website

A modern and responsive luxury car showroom website built using HTML, CSS, and JavaScript. AutoLux provides an elegant platform for browsing premium vehicles, viewing collections, and completing vehicle purchases through a sleek payment interface.

## ✨ Features

* Modern luxury-themed design
* Responsive layout for all devices
* Interactive car collection showcase
* Vehicle details and ordering system
* Payment interface integration
* Clean and user-friendly navigation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
